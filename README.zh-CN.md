**X/Twitter 视频下载器是一个 [Claude Code](https://claude.ai) 技能，自动下载 X/Twitter 帖子中的视频。** 提供链接即可保存视频到指定位置。

## 快速安装

```bash
git clone https://github.com/Lucasyao1985/x-video-downloader.git
```
